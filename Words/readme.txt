Princeton WordNet Plugin for UWN API
http://www.lexvo.org/uwn/api/


== DESCRIPTION ==

This plugin provides access to the Princeton WordNet 3.0 data
using the UWN API. It can be used in conjunction with other
multilingual plugins provided on the UWN API website.

  english_words_etymology.csv — Generated by the etymology pipeline (see below)

== ETYMOLOGY PIPELINE ==

Enrich the master lexicon with etymology data using the three-agent pipeline.
Output: Words/english_words_etymology.csv
Columns: word, definition, phonemes, etymology_root, origin_language,
         language_family, historical_context, confidence, source_agent

  # Fast offline pass (~15s, ~29k words enriched):
  python3 -m ingestor.etymology_pipeline --skip-wiktionary

  # Full pass with Wiktionary phonemes + etymology gaps (requires internet):
  python3 -m ingestor.etymology_pipeline

  # Test on a subset first:
  python3 -m ingestor.etymology_pipeline --limit 1000 --dry-run

  Agents (priority order, highest first):
    1. seed_database — 138 hand-curated words; never overwritten
    2. etymwn        — etymwn.tsv; ~22k words with root forms and language
    3. collins_hint  — parenthetical language hints in Collins definitions
    4. wiktionary    — async Wiktionary REST API; resumable via checkpoint

== IMPORT TO POSTGRESQL ==

After generating the master lexicon, load it into the words table:

  python3 -m ingestor.words_csv_importer \
    --path Words/english_words_master_lexicon.csv \
    --batch-size 2000

  Or import the enriched etymology file:
  python3 -m ingestor.words_csv_importer \
    --path Words/english_words_etymology.csv \
    --batch-size 2000
